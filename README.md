# Mangaudio: 用于帮助制作有声（动态）漫画的一系列工具

## 项目简介

Mangaudio 是帮助多人协作制作有声漫画的一系列工具集合项目。

- 🤖️标注：可选自动化
- ✍🏻️标注：无法自动化，需要人工
- 🚧️标注：未来预期可以自动化，但是短时间内必须人工进行

一个有声漫画的制作包含如下的流程：

1. 从漫画中提取文本
   1. 🚧️自动生成文本（OCR）
   2. ✍🏻️校对文本
   3. 🤖️文本翻译
2. 使用VITS利用文本生成语音
   1. 🤖️语音合成
   2. ✍🏻️音频校对，需要检查分句、语气等
3. 🤖️导出文本和音频
4. ✍🏻️视频的制作和导出
